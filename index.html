Project Overview

Snowboard Support System

This project aims to create a simple snowboard support system solution for helping beginners who may encounter injuries and struggle to identify areas for improvement on their own.

Our solution takes accelerometer input and exchanges data with python through the ESP32 microcontroller and bluetooth. The data is interpreted to give real-time alerts to the user as well as tips.
